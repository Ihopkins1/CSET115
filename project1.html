Triangle Generator Demo

This project demonstrates basic JavaScript programming with a simple triangle pattern generator. It shows my understanding of loops and string manipulation.

Live Demo

Click the button to add rows to build a triangle!

Source Code

Original JavaScript Code:

const prompt = require("prompt-sync")();

for(let line = "#"; line.length <= 7; line += "#") {
    console.log(line);
}

Web Implementation:

let currentRow = 0;

document.getElementById('generateTriangle').addEventListener('click', function() {
    currentRow++;
    
    if (currentRow <= 7) {
        // Generate the current row
        let line = "#".repeat(currentRow);
        
        // Get existing output and add new line
        let currentOutput = document.getElementById('triangleOutput').textContent;
        if (currentOutput === "Click the button to generate a triangle pattern!") {
            currentOutput = "";
        }
        
        document.getElementById('triangleOutput').textContent = currentOutput + line + '\n';
        
        // Update button text to show progress
        document.getElementById('generateTriangle').textContent = `Add Row ${currentRow + 1}`;
        
        // If we've reached the end, change button to reset
        if (currentRow === 7) {
            document.getElementById('generateTriangle').textContent = "Reset Triangle";
        }
    } else {
        // Reset the triangle
        currentRow = 0;
        document.getElementById('triangleOutput').textContent = "";
        document.getElementById('generateTriangle').textContent = "Add Row 1";
    }
});

Project Overview

This project was originally a simple console application that generates triangle patterns using loops. I converted it into an interactive web application to demonstrate my ability to adapt code for different platforms.

The web version adds user interaction, progressive display, and visual feedback while maintaining the core logic of the original program.
