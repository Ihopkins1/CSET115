Project Overview

This interactive math quiz game demonstrates advanced JavaScript programming including random number generation, game state management, and user interface control.

The game features multiple difficulty levels, two game modes, and real-time scoring with immediate feedback.

Tools Used

JavaScript HTML CSS Game Logic DOM Manipulation

Features

  • Three difficulty levels (Easy, Medium, Hard)
  • Score mode (20 questions) and Strike mode (3 strikes)
  • Real-time scoring system
  • Question skipping functionality
  • Responsive game interface
